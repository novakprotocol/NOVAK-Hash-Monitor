# Contributing Policy

Thank you for your interest in the NOVAK Protocol.

At this time, the project is not open to:
- code contributions
- feature requests
- architecture modifications
- pull requests
- forks intended for implementation

Why?

NOVAK is a patent-pending, safety-critical invention involving
cryptographic integrity enforcement. To preserve legal clarity and
prevent accidental licensing, contributions must remain restricted.
